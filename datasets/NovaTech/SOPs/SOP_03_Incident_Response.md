# SOP: Incident Response

## Purpose
Protocol for P1/P2 outages.

## Scope
Applies to all NovaTech employees.

## Responsibilities
HR, IT, and respective Department Heads.

## Step-by-Step Workflow
- 1. Alert received
- 2. War room created
- 3. Mitigation deployed
- 4. Post-mortem written

## Approvals
Requires minimum 1 manager approval.

*Note: Reference recent outages e.g., NOV-1088*
